Architecture Project

Stone Veil

An exploration of materiality and facade design through stone elements. This project investigates how parametric variations in stone patterns can create dynamic architectural expressions.

Project Type

Facade Design

Tools & Software

Rhino 3D, Grasshopper

Year

2024

Focus

Material Expression

Project Overview

Stone Veil explores how natural materials can be expressed through digital design. The project combines traditional stone craftsmanship with contemporary parametric methods.

Through iterative design processes, varying densities and patterns of stone elements create depth and visual rhythm, transforming a simple material into a sophisticated architectural language.
